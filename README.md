Proyecto de un gestor de tareas realizado en una página web para el ramo de Diseño Web + IA.

Lenguajes utilizados:
- HTML
- CSS
- Javascript

Proyecto basado principalmente en lo visto y aprendido en clases y en actividades autonomas, con apoyo de las herramientas ya mencionadas, además de investigación en internet en sitios como w3school, y la utilización de la IA como material de apoyo.
